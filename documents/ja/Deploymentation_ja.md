# 環境セットアップ

## ZANSINインストール前の準備

両マシンのIPアドレスを確認してください。これらのアドレスはZANSINコントロールサーバーからZANSINトレーニングマシンへの接続に使用されるため、グローバルIPアドレスである必要はありません。

## ZANSINコントロールサーバーへのログイン

まず、`zansin` ユーザーでZANSINコントロールサーバーにログインします。
ログイン後、`zansin` ユーザーのホームディレクトリにいることを確認してください。

## セットアップスクリプトのダウンロード

次に、リポジトリから `zansin.sh` スクリプトをダウンロードします：

```bash
wget https://raw.githubusercontent.com/ZANSIN-sec/ZANSIN/main/zansin.sh
```

このスクリプトが環境セットアッププロセスを処理します。

## セットアップスクリプトの実行

スクリプトのダウンロードが完了したら、実行権限を付与して実行します：

```bash
chmod +x zansin.sh
sudo ./zansin.sh
```
